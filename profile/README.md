# LaraPlugins.io — Laravel Plugin Health Directory

LaraPlugins is a directory and analysis platform for the Laravel
ecosystem. It indexes over 79,000 Laravel plugins and provides automated health
scoring, version compatibility checks, security advisory tracking, and semantic
search across the entire package landscape.

Built by a solo developer out of frustration with guessing whether a package was
well maintained.

## What the Platform Offers

- **Health Scores** — Automated analysis of commit activity, release cadence,
  test coverage, documentation quality, and social trust signals.
- **Version Compatibility** — Filter by PHP version and Laravel version to find
  packages that work with your stack today.
- **Security Advisories** — Track known vulnerabilities across the Laravel
  package ecosystem.
- **Semantic Search** — Natural language search powered by vector embeddings and
  reranking, not just keyword matching.
- **MCP Server** — AI-native API that lets tools like Claude and Cursor search
  and evaluate plugins programmatically (725k+ calls in the last 30 days).
- **Watchlists & Data Exports** — Monitor packages you depend on and export
  health data for your own analysis.
- **LLM-Ready Datasets** — Full ecosystem exports in `llms-full.txt` and JSONL
  formats for AI tooling.
